================================================================
识物比价购物 AI APP 助手 — 项目说明
================================================================
版本：3.0（2026-05-22）
目标：拍照识物 → 多平台比价 端到端 Demo（完整功能：Redis缓存 + Bull定时 + 图片审核）

================================================================
一、项目概述
================================================================

"识物比价购物助手"完整 Demo，验证以下核心流程：

  0. 用户注册/登录（手机号+验证码，JWT 双令牌）
  1. 客户端拍照或从相册选取图片（支持多图融合识别）
  2. 图片压缩后通过 Base64 上传至服务端
  3. 服务端调用豆包多模态大模型 API 识别商品属性（Key 空时 mock）
  4. 服务端在 PostgreSQL 商品库中匹配商品，返回各平台最新价格
  5. 客户端以卡片列表展示比价结果（含收藏按钮）
  6. 用户点击商品查看多平台历史价格趋势图（fl_chart 折线图 + 历史最低价标记）
  7. 用户输入自然语言筛选条件（多轮对话，conversationId 串联）
  8. 商品评价聚合与摘要生成（正面/负面标签 + 总结文字）
  9. 收藏夹管理（多清单 + 商品管理）
 10. 搜索历史自动保存
 11. 降价提醒设置、修改与查看（PATCH 更新目标价格）
 12. 收藏夹商品点击跳转商品详情
 13. 启动页静默登录检测 → 自动跳转首页或登录页

二、技术栈
================================================================

客户端：Flutter 3.x（Dart 3.x）
  状态管理       flutter_riverpod 2.5.1
  网络请求       dio 5.4.0（拦截器 + Token 自动刷新）
  路由导航       go_router 10.0.0
  图表展示       fl_chart 0.68.0
  图片选择       image_picker 1.2.2
  骨架屏         shimmer 3.0.0
  安全存储       flutter_secure_storage 10.x（Token 持久化）
  跨平台         Android / iOS / 鸿蒙（占位）

服务端：Node.js + Nest.js 10（TypeScript 5.x）
  HTTP 框架      @nestjs/core 10.x
  数据库         PostgreSQL（TypeORM 0.3.x + pg 驱动）
  AI 视觉+对话   豆包多模态大模型 API（OpenAI-compatible，单接口通吃）
  认证           @nestjs/jwt + @nestjs/passport + passport-jwt（双令牌）
  安全           Helmet + @nestjs/throttler（限流）
  密码加密       bcryptjs
  环境变量       dotenv
  推送           FCM（Demo 日志模式占位）
  容器化         Docker + docker-compose（PostgreSQL + Redis + App 一键启动）

六、关键设计决策
================================================================

  1. PostgreSQL + pg 驱动 — attributes JSONB, price DECIMAL(10,2), date DATE
  2. 豆包多模态 API 统一 — 图像识别+NLP 用同一接口，单 doubaoConfig 管理
  3. Mock 兜底 — Key 为空时识别返回 {运动鞋,Nike}，NLP 用 13 条关键词规则
  4. DTO 装饰器 — ValidationPipe whitelist 模式下所有 DTO 必须加 @IsString 等装饰器
  5. Kotlin 增量编译禁用 — Flutter pub 缓存(C:)与项目(D:)跨盘符，gradle.properties 设 kotlin.incremental=false
  6. 图片压缩 — image_picker maxWidth/Height=1024 + Q80 直接输出，无需二次编解码
  7. NLP 输出校验 — validateFilter() 白名单：sort(3种) shopType(3种) priceRange(min≥0 max≥0)
  8. 四状态 UI — loading / error / empty / data，AsyncValue.when() 全覆盖
  9. 栈式路由 — go_router push/pop（非 go），保留返回历史
 10. 权限处理 — 四种结果 success/cancelled/permissionDenied/error，永久拒绝弹 AlertDialog
 11. 禁止端侧 AI — 客户端不引入任何 AI SDK，AI 由服务端 REST API 提供
 12. Dio 拦截器 — 自动附加 Bearer Token + 401 自动尝试 refresh + 失败跳登录
 13. 种子预置 — 30 商品×3 平台×7 个月=630 价格 + 测试用户 + 默认清单 + 评价摘要
 14. search_history 自动保存 — recognition 成功后自动记录（userId 可选）
 15. 降价提醒弹窗 — detail_screen 中弹出输入目标价，调用 /price-alerts
